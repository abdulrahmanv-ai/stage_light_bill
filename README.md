# 🎉 منصة الأضواء - نظام الفواتير

## خطوات الرفع على Railway (5 دقائق فقط)

### الخطوة 1 — سجّل حساب GitHub
1. روح على: https://github.com
2. اضغط "Sign up" وسجّل حساب مجاني

### الخطوة 2 — ارفع الملفات على GitHub
1. بعد تسجيل الدخول، اضغط "New" (الزر الأخضر)
2. اسم المشروع: `stage-lights`
3. اضغط "Create repository"
4. اضغط "uploading an existing file"
5. ارفع هذه الملفات الثلاثة:
   - `index.html`
   - `server.js`
   - `package.json`
6. اضغط "Commit changes"

### الخطوة 3 — ارفع على Railway
1. روح على: https://railway.app
2. اضغط "Login with GitHub"
3. اضغط "New Project"
4. اختر "Deploy from GitHub repo"
5. اختر مشروع `stage-lights`
6. انتظر دقيقتين حتى يكتمل النشر ✅

### الخطوة 4 — احصل على رابطك
1. اضغط على اسم المشروع
2. اضغط "Settings"
3. تحت "Domains" اضغط "Generate Domain"
4. ستحصل على رابط مثل: `stage-lights-xxx.railway.app`

### الخطوة 5 — شارك الرابط 🎉
- افتح الرابط من أي جوال أو جهاز
- البيانات تُحفظ في قاعدة بيانات مشتركة بين كل الأجهزة
